- Disregard SSL certificates when installing pytest:
```
pip \
  --trusted-host pypi.org \
  --trusted-host pypi.python.org \
  --trusted-host files.pythonhosted.org \
  install pytest
```

- `cd ~/project/path/venv/ && source activate` = Activate virtual environment.
